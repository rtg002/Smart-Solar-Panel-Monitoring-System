☀️ Smart Solar Panel Monitoring System

An IoT-based system designed to monitor the real-time performance of solar panels using **Arduino UNO and ESP8266**, enabling efficient energy management and remote data access.

---

## 📌 Project Overview

This project focuses on building a **smart monitoring system for solar panels** that tracks environmental and electrical parameters in real time.

Using **Arduino UNO** and **ESP8266 Wi-Fi module**, the system collects and transmits data such as:
- 🌡️ Temperature
- 💧 Humidity
- ⚡ Voltage
- 🔌 Current

The data is sent to an **IoT dashboard**, allowing users to monitor solar panel performance remotely and improve efficiency through data-driven insights.

---

## ✨ Features

- Real-time solar panel monitoring
- IoT-based wireless data transmission
- Temperature and humidity sensing (DHT11)
- Voltage and current measurement
- Live dashboard visualization
- Continuous performance tracking
- Data logging for analysis
- Remote access via web/mobile

---

## 🛠️ Technologies Used

- Arduino UNO
- ESP8266 Wi-Fi Module
- DHT11 Sensor
- Voltage Sensor Module
- Current Sensor (ACS712)
- IoT Dashboard (ThingSpeak / Blynk / Web Server)

---

## ⚙️ Working Principle

1. Solar panel generates electrical energy.
2. Sensors measure:
   - Voltage and current output
   - Ambient temperature and humidity
3. Arduino processes sensor data.
4. ESP8266 transmits data via Wi-Fi.
5. Data is displayed on an **IoT dashboard** in real time.
6. Users can monitor performance remotely.
7. Logged data helps in improving solar efficiency and maintenance.
